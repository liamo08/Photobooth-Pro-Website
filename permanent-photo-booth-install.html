FAQ

Permanent Install
Common Questions

What's "kiosk mode"?

A locked-down full-screen mode. Guests can only interact with the booth UI; no access to the Windows desktop, file system, or other apps.

Can I manage the booth remotely?

Yes — BoothLedger Cloud lets you push template changes, check device status, and view live galleries from anywhere.

Does it support cashless payments?

Yes — pay-per-print or pay-per-AI-effect via Stripe-friendly endpoints. Configurable per location.

What if the machine crashes overnight?

Auto-recovery restarts the app. Daily 4am scheduled reboot keeps the booth fresh. Remote alerts notify you of repeated failures.

Can I run multiple permanent installs from one account?

Yes — each location is a device license. Manage all of them from one BoothLedger Cloud dashboard.
